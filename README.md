  ## Technologies
  This application uses the following technologies:
  * JavaScript
  * Node.js v18.12.1
  * Express
  * MySQL2
  * Sequelize
  * Environmental Variables ('dotenv')
  * Nodemailer
  * Gmail SMTP
  * Bootstrap CSS Framework
